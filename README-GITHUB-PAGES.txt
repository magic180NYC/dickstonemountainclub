# 딕소톤 마운틴 클럽 GitHub Pages 업로드 파일

이 폴더 안의 파일과 폴더를 GitHub 저장소 루트에 그대로 업로드하세요.

중요:
- 이 폴더 자체를 업로드하지 말고, 폴더 안의 내용물을 올리세요.
- index.html, archive.html, dashboard.html, app.js, styles.css, data 폴더, assets 폴더가 저장소 루트에 있어야 합니다.
- GitHub Pages에서 Source는 Deploy from a branch, Branch는 main/root로 두면 됩니다.

검수 상태:
- 산 데이터: 100개
- 코스 없는 산: 0개
- 총 코스: 304개

라이브 서비스 전에는 지도+후기 기반 초안으로 표시된 산의 실제 거리, 소요시간, 출발지, 편의시설을 산별로 최종 검수하세요.
