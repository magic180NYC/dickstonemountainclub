# 딕스톤 마운틴 클럽 최종 GitHub Pages 폴더

이 폴더 하나만 보관하면 됩니다.

GitHub Pages 업로드 방법:
- 이 폴더 자체를 올리지 말고, 폴더 안의 파일과 폴더를 GitHub 저장소 루트에 업로드하세요.
- index.html, archive.html, dashboard.html, app.js, styles.css, data 폴더, assets 폴더가 저장소 루트에 있어야 합니다.

최신 반영 내용:
- DS 로고를 이미지 로고로 교체
- 수락산 코스를 사용자 제공 이미지 기준 3개 코스로 반영
- 100개 산 데이터와 총 304개 코스 포함

검수:
- 산 데이터: 100개
- 코스 없는 산: 0개
- 총 코스: 303개
- 수락산 코스: 수락산역 원점회귀 코스, 장암역 석림사 최단 코스, 당고개역/수락산역 노원골 코스

주의:
- Deploy from a branch 방식에서는 .github/workflows 배포 파일이 필요 없습니다.
- GitHub 저장소에 이미 .github/workflows/pages.yml이 올라가 있다면 삭제하세요.
- 그렇지 않으면 GitHub Actions 배포가 별도로 실행되며 실패할 수 있습니다.
- 이 최종 폴더에는 GitHub Actions 워크플로는 포함하지 않았습니다.
